    安徽省人民政府办公厅关于分解省政府第三次全体会议确定的重点工作任务的通知

    (皖政办秘〔2008〕35号)




    有关市人民政府，省政府有关部门：

    按照中央和省委的决策部署，省政府第三次全体会议要求，在全面完成年初确定的各项既定目标任务的基础上，下半年要重点抓好8个方面工作。现将省政府第三次全体会议确定的8个方面重点工作任务分解并落实到有关市和相关部门。各有关部门要切实履行职责，精心组织，认真落实，务求实效。省政府将适时调度，逐项督查，确保工作落到实处。

    一、加快转变发展方式，推进自主创新和节能减排

    1. 启动合芜蚌自主创新综合配套改革试验区建设，塑造区域创新品牌，增强示范和带动效应。（省发展改革委、省科技厅、省发展研究中心、合肥市、芜湖市、蚌埠市）

    2. 大力实施金融改革创新特别是发展风险投资、股权融资等直接融资服务体系，探索设立自主创新产业投资基金、创业投资引导基金，着力打造金融服务平台。（省政府金融办、安徽银监局）

    3. 加快淘汰落后产能，加强企业技术改造，严格新上项目环境准入，推进城市污水处理厂、节能与资源综合利用、工业废水治理和二氧化硫综合治理等重点项目建设，努力实现全年节能减排目标。（省经委、省建设厅、省环保局）

    二、加强农业和粮食生产，着力增加农民收入

    1. 扎实推进水稻产业提升行动和玉米振兴计划，确保秋季农业丰收，全力争取全年粮食产量达到600亿斤。（省农委）

    2. 大力发展养殖业，有效组织劳务输出，加快发展农产品加工业，深入推进农业产业化“532”提升行动，大力发展各类专业合作经济，多途径增加农民收入。（省农委、省劳动保障厅）

    3. 抓住国务院编制《国家粮食安全中长期发展规划纲要》的重大机遇，认真把我省的规划编制好，争取国家支持。（省发展改革委、省农委、省粮食局）

    4. 结合新农村建设，谋划、储备和推进中低产田改造、灌区配套、中小水库除险加固、农村公路建设、农技推广服务体系和农产品质量安全体系建设等一批重大项目。（省发展改革委、省农委、省水利厅、省交通厅）

    三、更加重视投资和项目工作

    1. 大力提升“861”行动计划，抓前期项目落地、抓在建项目进度、抓竣工项目投产、抓储备项目谋划，把深谋后劲项目作为当务之急，全面提高项目工作水平。（省发展改革委）

    2. 大力发展煤化工、煤层气开发和替代能源，适时推进煤电上下游企业联合。充分利用长江岸线资源优势，围绕资源深度开发、延长产业链条，重点谋划和推进一批重化工业项目。（省发展改革委、省经委）

    3. 瞄准生物、电子信息、新材料、新能源等产业，加紧谋划和推进一批重点项目。加快火灾、煤矿安全、道路交通、重大自然灾害防治等公共安全方面技术成果的产业化步伐，谋划出一批重大项目。抓紧储备推出一批余热及垃圾发电、瓦斯综合利用、环境监测服务、环保装备生产等我省有一定基础、市场潜力巨大的项目。（省发展改革委、省经委、省科技厅、省公安厅、省环保局）

    4. 瞄准现代服务业发展方向，以生产性服务业为突破口，围绕物流、会展、商务服务、文化创意等推出一批实实在在的项目，提升我省服务业发展水平。（省发展改革委、省经委、省商务厅、省文化厅）

    5. 完善项目推进机制，加大项目前期经费投入，创新项目生成方式，拓展项目来源。（省发展改革委）

    四、深入推进改革开放，增强经济发展活力

    1. 加大农村综合改革、林权制度改革等推进力度，巩固成果，及时总结推广典型经验。（省农委、省林业厅）

    2. 大胆探索土地经营权流转、农房抵押贷款等改革，抓紧启动和扩大试点工作。（省国土资源厅、省建设厅、省政府金融办、安徽银监局）

    3. 推进城乡一体化综合配套改革试点，芜湖市、马鞍山市、铜陵市先行试点，探索路子，为全省创造和提供新鲜经验。淮北市改革方案要抓紧上报省政府。（省发展改革委、芜湖市、马鞍山市、铜陵市、淮北市）

    4. 围绕承接产业转移，加强载体和平台建设，改进招商方式和方法，提升引资工作的针对性和实效性。（省发展改革委、省商务厅）

    5. 争取合肥出口加工区早日通过国家批准。（省发展改革委、省商务厅、合肥市）

    6. 利用合肥、芜湖、安庆市被认定为“加工贸易梯度转移重点承接地”的机遇，加速提升承接条件，努力吸引更多的加工贸易转移。（省发展改革委，合肥、芜湖、安庆市）

    7. 抓紧研究，积极推进在省内模拟建设进出口货物口岸国际物流管理中心，深化通关模式改革，解决出入境货物直接放行问题。（省商务厅）

    五、进一步优化环境，推动非公经济和中小企业加快发展

    1. 针对当前非公经济和中小企业发展面临的准入难、融资难、用地难等突出问题，研究提出一些务实管用的办法，切实推动。出台实施进一步促进非公经济发展的意见。（省经委）

    六、完善金融服务体系，大力拓宽融资渠道

    1. 增加对“三农”、中小企业、科技创新与节能环保、房地产等领域的贷款，加大对有订单、信誉良好企业的信贷支持。（省政府金融办、安徽银监局）

    2. 抓好银企对接月签约项目的兑现落实，提高履约率。（省发展改革委、省政府金融办、安徽银监局）

    3. 深化农村信用社改革，重点推进农村合作银行组建，积极开展村镇银行、小额贷款公司试点工作；加快徽商银行省内布点和增资扩股工作，认真做好民生银行筹建合肥分行、华夏银行来皖设立分支机构的服务工作。（省政府金融办）

    4. 大力发展直接融资，积极培育上市后备资源，争取再有一批企业上市和再融资。通过发展国内信用证、保函和银行承兑汇票等业务，增强企业融资能力。（省政府金融办）

    七、着力解决民生问题，保持社会和谐稳定

    1. 加强物价调控和市场监管,落实好城市低保提标和成品油价格调整后的补贴政策,逐步完善物价上涨与补贴标准联动机制。（省财政厅、省物价局）

    2. 扩大就业，做好困难家庭、零就业家庭和高校毕业生的就业服务。（省劳动保障厅、省教育厅）

    3. 继续抓好18项民生工程的实施，加强督查指导和跟踪评估，确保按时完成任务。（省财政厅）

    4. 以“三清”、“三改”、“四灭”为重点，深入开展爱国卫生运动，进一步加强传染病防治工作。（省卫生厅）

    5. 密切关注信访群众诉求和倾向性、苗头性问题，积极预防和妥善处置群体性事件，坚决防止和避免因应对和处置不当不力而扩大事态，造成恶劣影响。（省信访局）

    6. 抓好社会治安防控和应急值守工作，加强对公共聚集场所、重点关键部位的监测监控，防止发生影响稳定的重大事件。（省公安厅、省司法厅、省政府应急办）

    7. 严格落实领导责任制，强化重点行业领域的隐患排查和整治，杜绝特大事故发生。（省安全生产监管局）

    八、全力抓好对口支援，确保完成年度任务

    1. 10月底前完成民房、水利、校舍、村道恢复和社会福利设施建设等5项应急援建任务，年内基本完成各类规划和实施方案编制工作。（省发展改革委）

    2. 积极做好提供人才智力援助、增加就业岗位、尽快恢复旅游经济、帮助招商引资、促进产业发展等工作。（省发展改革委）

    二○○八年八月十七日
